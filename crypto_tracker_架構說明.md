# 加密貨幣自動追蹤系統 — 完整架構說明

## 專案概述

每天早上 09:00（台灣時間）自動執行，追蹤 BTC / ETH / XRP 三個幣種，
透過技術指標分析與 AI 建議，推播到 Telegram，完全免人工操作。

---

## 出場策略說明

進場後依照 ATR 分三批出場，鎖定利潤同時讓最後部位追趨勢：

```
進場（站穩壓力關卡）
   │
   ├─ 停損：進場價 - 1.5xATR（硬停損，觸發全出）
   │
   ├─ TP1 達到 → 出 1/3 部位，停損移至成本價（保本）
   │
   ├─ TP2 達到 → 再出 1/3，停損移至 TP1（鎖利）
   │
   └─ TP3 達到 → 剩餘部位改用移動停利
                 （價格從高點回落 1.5xATR 即出場）
```

## 排程設定

```yaml
cron: '0 1 * * *'
# UTC 01:00 = 台灣時間 09:00，每天執行一次
```

如需調整時間，修改 `.github/workflows/btc_breakout.yml` 裡的 cron 設定：
- 每天 06:00 台灣時間：`0 22 * * *`（前一天 UTC 22:00）
- 每天 08:00 台灣時間：`0 0 * * *`（UTC 00:00）
- 每天 09:00 台灣時間：`0 1 * * *`（UTC 01:00）✅ 目前設定
